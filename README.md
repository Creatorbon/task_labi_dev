# Task Labi.dev

# Start App
Clone repo, install, cd into folder and run:
```git
npm run start
```
or
```git
yarn run start
```
You can build develop version of app with source map
```git
npm run build:dev
```
or
```git
yarn run build:dev
```
To build production app version
```git
npm run build
```
or
```git
yarn run build
```
